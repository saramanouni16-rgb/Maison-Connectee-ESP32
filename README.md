# 🏠 Maison Connectée – ESP32

## 📌 Description
Smart Home monitoring system using ESP32 with gas and rain detection.

## ⚙️ Technologies
- ESP32
- MicroPython
- Firebase Realtime Database
- LCD I2C
- MQ-2 Gas Sensor
- Rain Sensor

## 🚨 Features
- Gas detection alert
- Rain monitoring
- LCD display
- Firebase cloud monitoring
- Buzzer & LED alarm
